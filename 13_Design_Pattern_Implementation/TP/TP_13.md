# Deskripsi Program
Jadi Design ini adalah Observer Design, yang dimana router.events adalah sumber event, handleRoute adalah callback yang “mendengarkan” event, saat navigasi selesai, Next.js memicu event routeChangeComplete dan ketika event itu muncul, handleRoute akan dijalankan dan memanggil loadSchedules()

Desain ini juga memakai prinsip:

+ Subject/Event Source: router.events
+ Observer/Listener: handleRoute
+ Event: routeChangeComplete

Jadi alur desainnya adalah: “saat ada event tertentu terjadi, lakukan aksi tertentu.”
