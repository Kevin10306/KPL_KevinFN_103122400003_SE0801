## Deskripsi Program
Program ini merupakan implementasi fungsi toNumberArray menggunakan JavaScript dengan runtime Node.js yang dibuat untuk mengkonversi input berupa string atau array menjadi larik angka.
Fungsi toNumberArray menerima satu parameter number dan melakukan pengecekan tipe input dalam dua kondisi. Pertama, jika input bertipe string, maka string tersebut akan dipecah berdasarkan karakter koma (,), kemudian setiap elemen dibersihkan dari spasi menggunakan .trim(). Kedua, jika input berupa array, maka array tersebut langsung digunakan tanpa transformasi awal. Apabila input bukan string maupun array, fungsi akan melemparkan TypeError dengan pesan "Input harus berupa string atau array"
