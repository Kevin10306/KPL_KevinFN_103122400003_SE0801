Buatlah sebuah fungsi bernama fizzBuzz yang menerima input larik (array) dan mengembalikan deretan bilangan dan "Fizz" untuk kelipatan 2, "Buzz" untuk kelipatan 7, dan "FizzBuzz" untuk kelipatan 14. Beri nama berkas program sebagai tm.js dan taruh di direktori TM.

Kode Sumber [tm.js]()




Kode ini berfungsi untuk memproses array angka dan mengubahnya menjadi string berdasarkan aturan pembagian angka 2 dan 7. Program ini memeriksa angka jika angka habis dibagi 14 maka diganti dengan teks "fizzBuzz". Jika angka habis dibgai 7 maka diganti teks "Buzz". Jika angka habis dibagi 2 maka diganti teks "Fizz"
