## Deskripsi Program
Program ini adalah sebuah REST API sederhana berbasis Node.js dan Express yang mengimplementasikan permainan tebak angka. API menerima nama pemain dan tebakan angka, lalu memberikan respons apakah tebakan tersebut benar, terlalu tinggi, atau terlalu rendah, menggunakan runtime node.js, dan framework express.js dengan endpoint "Post" yang akan mengirim tebakan
