## Deskripsi Program
Disini kita melakukan tukar uang dari IDR(Indonesia) ke CHN(Tiongkok), dengan menggunakan Intl.NumberFormat yang bisa Memformat angka, mata uang, persen, dan satuan pengukuran, dan kita gunakan API dengan Url seperti ini: https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api@latest/v1/currencies/idr.json, API ini sudah menyediakan data kurs nya, kita hanya ambil nilai yang di butuhkan
