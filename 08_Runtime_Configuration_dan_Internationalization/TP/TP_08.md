## Deskripsi Program
Mengambil waktu saat ini objek Date dibuat tanpa argumen sehingga otomatis menyimpan tanggal dan waktu pada saat program dijalankan lalu memformat tanggal (intl.DateTimeFormat) digunakan dengan locale 'id-ID', beserta opsi tampilan nya:
Weekday: 'long' untuk nama hari secara lengkap
Year: 'numeric' untuk tahun dalam angka
Month: 'long' untuk nama bulan secara lengkap
Day: 'numeric' untuk tanggal dalam angka
lalu untuk menampilkan hasil dengan metode .format() dipanggil dengan objek waktu, lalu hasilnya di cetak ke konsol menggunakan console.log
